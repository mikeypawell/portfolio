Voltage Cities

HTML5, CSS3, jQuery, Ruby on Rails, Bootstrap, Javascript, Google Maps API, and Twillio API

A web application to help musicians discover venues in new cities and share their experiences with like minded musicians. Beware....our instructions for this project were to build a 90's style website complete with radical colors, prints, and even a hit counter! The website is so awesome it hurts.
